moodle-block_countingdown
======================

Description:
------------
Countdown Block

This block displays a counting down clock.

It should work with all themes.

**Installation:**
Download, extract, and upload the "countingdown" folder into moodle/blocks/

Supported Moodle versions:
--------------------------
I have tested plugin on clean install of Moodle 3.0
